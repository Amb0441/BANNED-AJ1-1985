# BANNED // AJ1 1985 Archive

A premium, interactive digital archive and academic case study exploring the history, marketing, and cultural impact of the 1985 "Banned" Air Jordan 1. 

## 📖 Overview
This project serves as a technical demonstration of modern frontend development, UI/UX design, and the integration of Generative AI in asset production. It features an editorial-style layout, smooth scroll animations, and a fully responsive design.

## ✨ Features
* **Custom Interactive Cursor:** Unique dot and tracking ring that expands when hovering over clickable elements.
* **Cinematic Hero Section:** Slow-zoom AI-generated background with custom scanline and noise overlays.
* **Scroll Reveal Animations:** Content dynamically fades and slides into place using the Intersection Observer API.
* **Responsive Design:** Fully optimized for desktop, tablet, and mobile with swipeable navigation and edge-to-edge mobile imagery.
* **Modern Typography:** Utilizes 'Bebas Neue' and 'DM Sans' for a bold, editorial aesthetic.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Custom variables, CSS Grid/Flexbox, media queries, and complex keyframe animations.
* **Vanilla JavaScript:** DOM manipulation, custom cursor math, and scroll intersection observing (no external libraries).

## 🚀 How to Run Locally
This project uses standard web technologies and requires no build tools or package managers.
1. Clone or download the repository to your local machine.
2. Open the project folder.
3. Simply double-click `index.html` to open it in your default web browser, or use a tool like VS Code's "Live Server" extension.
